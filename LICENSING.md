# Licensing

## Adapter Layer

`codex-trusted-mode` is MIT-licensed.

That applies to:
- the local hardening engine
- the Codex request-normalization layer
- the PDP adapter code in this repo
- the examples and verification scripts in this repo

## Governance Runtime

The SDE enterprise runtime used for the paid governed mode is proprietary commercial software.

The MIT license for this repo does not grant rights to:
- the SDE enterprise PDP
- signed enterprise policy packs
- commercial entitlements or enterprise release artifacts shipped from the SDE product

## Practical Split

- free use of this repo: local hardening and integration development
- paid use of the governed SDE runtime: separate commercial license or written permission required

## Recommendation

Keep this split. It is the right commercial model here:
- permissive adapter layer for adoption
- proprietary governance runtime for the commercial offering
