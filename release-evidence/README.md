# Public Evidence Notes

This directory keeps a small public evidence subset for the Codex controlled-rollout boundary.

Included here:
- summarized validation results
- normalized native-hook evidence
- observed tool-surface summaries
- templates used to regenerate the public evidence set

Not kept here in git:
- raw local session transcripts
- ad hoc VM artifacts
- large intermediate capture files
- environment-specific scratch files

If you need to regenerate evidence locally, use the scripts under [`../scripts`](../scripts) and review the controlled-rollout boundary in [`../COMPATIBILITY_MATRIX.md`](../COMPATIBILITY_MATRIX.md).
